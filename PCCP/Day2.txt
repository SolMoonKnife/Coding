#. 알고리즘 평가 : 효율성과 정확성.
정확성 : 테스트 케이스 통과도
효율성 : 시간/공간 복잡도 - 특히 시간 복잡도가 중요한 편.
    *시간 복잡도 : 제한 시간 이내 최소값 지향 (입력 ~ 출력 사이 시간)
    *공간 복잡도 : 메모리 제한 이내 최소값 지향
2. 재귀 구조 - 종료 조건(Base)가 필수적이다. 점화식을 통해 자기 자신을 호출한다.
    즉 자기 자신을 호출하다가 특정 조건(인자)을 만족할 경우 자기 자신을 호출하지 않고 특정 값을 반환하는 종료 조건 필요.

이진 탐색을 재귀적으로 구현할 수 있는 아이디어
처음에 중간을 찾아서 본 다음에 크고 작음에 따라서 절반씩 나누어 재귀적으로 호출하다가 반환하면 되지 않을까?

def func(list, value):
    length = len(list)
    pos = length//2
    if list[pos] < value:
        return func(list[pos:], value)
    elif list[pos] > value:
        return func(list[:pos], value)
    else:
        return list[pos]
