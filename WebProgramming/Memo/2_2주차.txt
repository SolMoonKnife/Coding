<태그>
값은 보통 큰따옴표 안에 쓴다.
큰따옴표 안쓰면 값 사이에 공백이 허용되지 않음

여는 태그와 닫는 태그는 일반적인 코드처럼 들여쓰기 순서에 따라
포함 관계를 정확히 해야함. 자바스크립트는 DOM생성을 하는데,
태그 순서가 클리면 정확히 만들어지지 않는다.

시작 태그만 있는 <br> 태그의 경우 <br/>처럼 여닫기 한 번에 표현
-> DOM model에 도움을 준다. 또는 업계 표준이다.

★태그 이름은 대소문자를 구분하지 않는다.

타이틀 태그 <title> ... </title>
메타 태그 <meta> 보통 html문서에 대한 정보를 표시함
<meta charset="UTF-8"> 문서의 인코딩 지정 -> 구라치면 깨진다.

<p> ... ... ... </p> 문단을 나눠주는 태그

<script>: 자바스크립트
메타태그는 대부분 <head>내부에 작성된다. 단, 스크립트는 바디 내부에도 들어감.

<base>: 가장 베이스가 되는 웹페이지 주소를 전역으로 지정해두고 사용 가능
<head>
	<base href="http://www.mysite.com/score/">
</head>
<a href="math.html">수학</a>
<a href="science.html">과학</a>
-> mysite의 score에 들어가서 math.html 또는 science.html을 열 수 있음

한 화면을 3개의 프레임으로 구분해서 표시하려면?
전체 틀이 되는 것까지 포함하여 총 4개의 html문서가 필요함.
서브 프레임은 base target으로 메인 프레임을 지정한다.

하이퍼링크의 중요한 두 가지 속성
href, target
<a href="URL"|"URL#anchor"|"#anchor" target="window_name">
    text or image
</a>

target
_self: 링크를 현재 창(현재 브라우저 탭)에서 엽니다. 이것이 기본값입니다.
_blank: 링크를 새로운 브라우저 창이나 탭에서 엽니다.
_parent: 링크를 부모 프레임에 엽니다. 부모 프레임이 없으면 _self와 같이 동작합니다.
_top: 링크를 최상위 창(가장 높은 수준의 프레임 또는 창)에 엽니다.
사용자가 정의한 프레임의 이름: 링크를 해당 프레임에 엽니다.

하이퍼링크 텍스트의 색 (# CSS3를 이용하여 링크 색 지정 가능 )
링크 텍스트(standard link) 처음 색 – 밑줄과 함께 blue
방문 후 링크(visited lnk) 색 – purple
마우스로 링크를 누르고 있는 동안(active link) – red
